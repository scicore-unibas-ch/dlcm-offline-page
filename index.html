LabKey DLCM server instance phased out


The Swissuniversities project DLCM has reached the end of its funding cycle. The proof-of-concept "National LabKey Server" deployed in DLCM context has thus been shut down.

For any question related to the "National LabKey Server" project, to request access to LabKey at University of Basel, or for any question related to using LabKey in academic research, please contact sciCORE, the scientific computing center at University of Basel.

For more information:

  • Data Life Cycle (DLCM) project
  • LabKey
